Documentation

I do not have a pet and would basically just made up a pet for myself. I don't really know much about pets so I asked some friends about what their pets would do in certain situations and guessed for the rest. The idea for my state transition was just to make it as easy as possible to understand and implement effectively. If statements are always really easy to understand.Every 20secs the pet spontanously changes state in a cycle. My code is heavly commented to help programmers add upon my work. I assign a value to each of my states and use an integer "currentState" to keep track of what state I am in. 1 = relaxing, 2 = hungry, 3 = happy, 4 = angry, 5 = sleepy
